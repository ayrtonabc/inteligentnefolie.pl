# Tailandia CMS

Panel de administración para gestionar el contenido del sitio web de Tailandia.

## Características

- 🔐 Sistema de autenticación
- 📊 Dashboard con estadísticas
- 🏗️ Gestión de proyectos
- 💰 Gestión de inversiones
- 📧 Gestión de mensajes de contacto
- 🎨 Interfaz moderna con Tailwind CSS

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor se ejecuta en: http://localhost:5174

## Construir

```bash
npm run build
```
